# CAT: ULTIMATE PURRRVIVOR

Kate Steele, Carrie Sargent, Zach Riggle

## Mac Install:

    Download Mac Build.zip
    Extract the .zip file on your system.
    Launch Cat-Ultimate Purrvivor app to play!

## Windows Install:

    Download Survivorlike-Windows.zip
    Extract the .zip file on your system.
    Open Survivorlike-Windows
    Double-click Survivorlike.exe to launch the game!

## Escape the Rats!

#### You are a genetically engineered CAT on the run from genetically engineered RATS! Run, fight, and PURRRVIVE!!!!!!

## Play the Game!

* Use arrow keys/ WASD to move
* Attack the rats with lasers! (Automatically fires)
* Find powerups to heal and boost!
* Kill rats to level up! 
* Survive for as long as you can! Don't let those dirty rats win!
